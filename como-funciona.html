Como funciona o Chatzinho

O Chatzinho é um site de conversas anônimas criado para promover interações simples, respeitosas e seguras.

Passo a passo

  1. Escolha entre Chat Casual ou Chat +18.
  2. Respeite as regras do site.
  3. Converse com responsabilidade.
  4. Encerre a conversa se algo não lhe agradar.

Dicas importantes

  • Não compartilhe dados pessoais.
  • Não envie conteúdos ilegais.
  • Use a denúncia quando necessário.

O Chatzinho existe para proporcionar boas experiências a todos.
